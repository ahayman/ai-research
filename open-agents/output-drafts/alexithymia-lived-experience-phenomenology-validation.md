# Self-Validation Notes: Lived Experience and Phenomenology (Cat 12)

## Validation purpose

This document records spot-checks of the citations and claims in the concept-research file before downstream agents (diagrams, explanation, webpage) build on it. The category is phenomenological — most of its content is derived from clinical writing, qualitative research, and first-person accounts rather than RCTs — so the validation focus is on:

1. Citation accuracy (real papers, correct attribution).
2. Faithful representation of what each source actually claims.
3. Honest framing of the epistemic status of phenomenological vs. quantitative evidence.
4. Application of universal corrections from the validation supplement.

## Contested claims flagged

The concept-research file lists three contested claims explicitly so the explanation agent hedges appropriately:

1. **The autistic-affirming reframe** as descriptive cognitive-style account vs. complete displacement of deficit framing — the explainer should hold both.
2. **The "failed therapy patient" iatrogenic-shame narrative** as well-documented clinical-consensus pattern, not as RCT-grade finding.
3. **Type-specific phenomenology mapping to Bermond Type I/II** as useful descriptive vocabulary, not as validated diagnostic typology.

## Open questions / not directly addressed

1. The Calsius 2015 IPA study sample size and demographics are not explicitly stated in the concept-research file (typical of IPA: the sample is in the dozens, mostly women, fibromyalgia diagnosis). The explanation agent should hedge "small qualitative sample, single clinical population, peer-reviewed but not high-impact-journal" to match the validation supplement's framing.

2. The first-person literature is unevenly attributed in some places ("the McKenna Medium essay" without specific URL). The explanation agent can preserve this level of citation specificity since the concept-research file is honest about what's grey literature.

3. The phenomenological types described (affectless / feelings-present-but-illegible / hypervigilant-with-mislabelling) are the concept-research file's synthesis of multiple lineages. Bermond's published Type I/II is a proper subset. The explanation agent should make clear that the three-type description is descriptive synthesis, not Bermond's published taxonomy.
