# Validation: Semasiographic vs. Glottographic Systems

## Validation Summary

This document is strong. The core theoretical framework — Gelb's taxonomy, the semasiographic-glottographic distinction, and the analysis of functioning semasiographic systems — is accurately presented and well-sourced. The document appropriately notes that Gelb's teleological claim is now contested. The analysis of mixed systems and the sign language counter-argument are nuanced and well-supported. One minor historical date needs qualification.

## Claim-by-Claim Verification

### Gelb's Taxonomy (1952)

| Claim | Rating | Notes |
|-------|--------|-------|
| Ignace Gelb systematized the semasiographic/glottographic distinction in *A Study of Writing* (1952) | CONFIRMED | Gelb coined "semasiography" in his 1952 work, deriving it from Greek *semasia* (meaning) and *graphe* (writing). |
| Gelb positioned semasiography as a "forerunner" stage | CONFIRMED | Gelb classified semasiography as a precursor or "forerunner" to true writing. |
| Gelb's teleological claim: writing evolves from semasiographic to phonographic | CONFIRMED | He suggested scripts evolve in a single direction: logographic → syllabic → alphabetic. |
| The teleological view is now contested | CONFIRMED | Multiple sources note this has been "criticized as overly simplistic" and reflecting "Western alphabetic bias." |
| Gelb's classification: semasiography → word-syllabic → syllabic → alphabetic | CONFIRMED | This linear progression model is well-documented as Gelb's position. |
