## Systemic Issues

### Issue 1. The "representative citation" shortcut is the weakest part of the chunk

All four flagged representative citations turn out to have problems of exactly the kind the build agent's own hedge should predict. Two of the four (Shaw et al. 2022; Douglas & Sedgewick 2024) point to real papers that the build agent already flagged with hedges but got wrong in bibliographic details (year for Shaw; first name for Douglas). Two of the four (Lim & Magiati 2022; Doggett et al. 2021) point to no locatable paper at all and should be treated as fabrications pending further evidence.

This is a concentrated failure mode. The fix is mechanical: delete or replace each of the four flagged citations with either the correct bibliographic record for the intended paper or a proper [COMMUNITY KNOWLEDGE] flag if the underlying claim is supported by autistic community accounts rather than by a specific paper.

### Issue 2. The citations are concentrated in Cat 24 and its practical/autism files

Cat 23 is cleaner than Cat 24 in citation practice. Cat 23's three research files use the same canonical theoretical literature and the same verified autism research, and the citations in Cat 23 are essentially error-free apart from the Pearson Rees Forster year slip. The four flagged citations and the fabricated ones are all concentrated in Cat 24's practical and autism files. The asymmetry suggests the build agent ran out of verification capacity by the time it finished the Cat 24 files, which matches the build order.

### Issue 3. The hedging language within the text is slightly looser than the bibliography entry hedging

Two of the representative-citation entries in the Cat 24 autism file's bibliography list the hedge explicitly ("[Representative citation; check current reference list for exact details.]"). This is better than nothing but is still inadequate for a file that is going to be cited by 40 manipulation sub-hub pages. The manipulation sub-hub pages will not carry the hedge forward; they will link to Cat 24 as authoritative. Any citation that is uncertain enough to need a "check current reference list" flag should not be cited in a file that loads downstream references.

### Issue 4. Douglas & Sedgewick first-name error

The Cat 24 autism file line 159 reads "Recent work by Patricia Douglas, Felicity Sedgewick, and colleagues". The correct first author is **Sarah Douglas**. The Cat 23 autism file and Cat 23 main file give "Douglas, S. & Sedgewick, F." and Douglas, S. which is correct. The error is isolated to one line of Cat 24 autism but should be corrected before the file is relied on by downstream pages.
