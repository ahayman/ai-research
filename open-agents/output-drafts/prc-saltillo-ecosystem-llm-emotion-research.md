## 1.2 TouchChat HD with WordPower -- Ecosystem

### 1.2.1 Core App
- **Platform:** iOS only (iPad, iPhone, iPod Touch)
- **Price:** ~$299.99 (one-time purchase from App Store)
- **Trial:** TouchChat Discover -- free 30-day trial app
- **Vocabulary System:** WordPower (designed by SLP Nancy Inman) -- core word + word prediction approach
- **Voices:** Inclusive Voices by Almagu (20+ diverse voices -- child, teen, adult, senior), Acapela voices, Apple system voices
- **Languages:** English (US, UK, AU, CA), Spanish, Canadian French; bilingual files removed in v2.52 (Feb 2025), replaced with per-language vocab files and code-switching capability

### 1.2.2 Companion Tools & Services
- **TouchChat Discover:** Free trial version for evaluation before $300 purchase
- **Vocabulary Builder:** Built into app; temporarily limits visible words on screen for teaching specific motor plans; SLPs can show/hide words without deleting them (visually hide/show added in 2026.1)
- **Word Finder:** Helps locate where words are within vocabulary page sets
- **Realize Language:** Subscription analytics platform ($9.95/year) -- see Section 1.5.1
- **Data Logging:** Built-in; can export to Realize Language or manually export (added v2.50)

### 1.2.3 Hardware Integration
- Pre-installed on NovaChat devices and Via devices with TouchChat software
- Via Pro with TouchChat supports direct touch, switch scanning, and other iOS access methods
- TouchChat Express is a Saltillo hardware device running TouchChat
- Inclusive Voices by Almagu included by default on all hardware devices

### 1.2.4 Backup & Sync
- **Cloud Sync:** Vocabulary files are NOT automatically synced across devices
- **Backup Methods:** iCloud Drive, AirDrop, Dropbox, Google Drive, iShare
- **User Backup/Restore:** Menu > Application Settings > System > User Backup/Restore -- backs up all user settings, pronunciation exceptions, abbreviations, word prediction data, and all profiles
- **File Format:** .ce format (as of recent updates)
- **Key Limitation:** Manual transfer required between devices; vocab files specific to TouchChat (won't open in LAMP or Unity)

### 1.2.5 User Satisfaction & Pain Points

**Overall Rating:** 3.92/5 stars on App Store (100 ratings)

**What Works Well:**
- Robust, customizable vocabulary system
- WordPower's core word + prediction approach praised by many SLPs
- Intuitive for users who learn it well
- "Son has used it for several years -- robust, customizable, and user-friendly"

**Major Complaints:**
1. **Documentation/Usability:** "Figuring out how to use the app is very difficult"; broken links to support docs; support docs not user-friendly
2. **Custom Vocabulary Transfer:** Upgrading from free trial to full $300 version downloads a new app with no apparent way to transfer customizations -- users must start over
3. **Technical Bugs:** iPad/app updates introduce bugs where app opens but icons don't respond; requires delete/reinstall and reimport of customized files
4. **Layout Inconsistency:** Vocabulary layout not consistent across levels -- users progressing to higher levels must relearn motor patterns as button locations change
5. **Price Shock:** $299.99 is very expensive, especially when bugs or transfer issues occur
6. **v2.52 Bilingual Confusion:** Major glitches in Feb 2025 update (2.52.0) -- app crashes, vocab builder lists lost, customized files lost; partially fixed in 2.52.1 (March 2025)

Sources: [TouchChat App Store](https://apps.apple.com/us/app/touchchat-hd-aac-w-wordpower/id412351574), [TouchChat Home](https://touchchatapp.com/), [OMazing Kids AAC Blog](https://omazingkidsllc.com/)

## 1.3 LAMP Words for Life -- Ecosystem

### 1.3.1 Core App
- **Platform:** iOS only (iPad, iPhone)
- **Price:** ~$299.99 (one-time purchase)
- **Trial:** LAMP Words for Life Discover -- free 30-day trial (launched April 2025)
- **Vocabulary System:** Unity language system + LAMP (Language Acquisition through Motor Planning) therapeutic approach based on Minspeak
- **Therapeutic Basis:** LAMP is a therapeutic approach based on neurological and motor learning principles, designed for individuals with autism and other developmental disabilities
- **Voices:** Inclusive Voices by Almagu, Acapela, Apple voices
- **Languages:** English (US, UK, AU, CA), Spanish, Canadian French

### 1.3.2 Companion Tools & Services
- **LAMP Words for Life Discover:** Free 30-day trial app
- **Vocabulary Builder:** Built-in; same functionality as TouchChat -- temporarily limits words for focused teaching
- **Word Finder:** Locates words within vocabulary layouts
- **Realize Language:** Same $9.95/year analytics platform
- **Data Logging:** Built-in with manual export (added v2.50); export via Menu > Settings > Data Logging > Export Log
- **AAC Language Lab:** Lite-tech LAMP boards can be printed from this resource for use at home/school
- **Minspeak Academy:** Partnership with Semantic Compaction Systems (SCS) for Minspeak language system education

### 1.3.3 Hardware Integration
- Available on Accent devices
- Recently added to NovaChat devices
- Via Pro with LAMP Words for Life configuration available
- Compatible with third-party switches (added in 2025.4.1)

### 1.3.4 Backup & Sync
- Same infrastructure as TouchChat -- no automatic cloud sync
- Manual backup via iCloud Drive, AirDrop, Dropbox, Google Drive
- **Critical Note:** When using on multiple devices (iPhone + iPad, or school setting), all devices must run the same app version (2025.4.1+) to transfer vocabulary files seamlessly
- Vocabulary files are app-specific (LAMP files won't open in TouchChat or Unity)
- Comes with 3 locked English vocabulary files and 2 locked Spanish/English files; must copy before customizing

### 1.3.5 User Satisfaction & Pain Points

**What Works Well:**
- LAMP motor planning approach is highly valued by autism-focused SLPs
- Consistent motor patterns for core vocabulary
- Strong theoretical foundation in motor learning

**Major Complaints:**
1. **Inappropriate Images:** Parents report AAC item images they consider inappropriate for children
2. **Limited Social Content:** Clinicians and teachers want more social vocabulary content
3. **Usability Issues:** Violations of design guidelines in areas of aesthetics, user errors, controls, and customization
4. **v2.52 Update Problems:** Same issues as TouchChat -- crashes, lost files, confusion about bilingual/multilingual file changes
5. **Transfer Confusion:** Student transitioning from TouchChat to LAMP became confused about button interactions, leading to longer button strings and OCD-like repetitive behavior
6. **Parent vs SLP Disconnect:** Different perspectives between parents and SLPs about app functionality and features

# TOPIC 2: LLM / AI USAGE

## 2.1 PRC-Saltillo AI Features (2026.1 Update)

### Overview
PRC-Saltillo introduced its first AI-powered features in the **2026.1 app update**, available across all four iOS apps:
- LAMP Words for Life
- TouchChat (with WordPower)
- Unity AAC
- Dialogue AAC

### Technology: Apple Intelligence
- **Underlying Technology:** Apple Intelligence (Apple's on-device AI platform)
- **NOT ChatGPT, Claude, or Gemini** -- uses Apple's built-in AI framework
- **Hardware Requirements:** Specific iOS hardware running iOS26 (requires Apple Intelligence-capable devices -- iPhone 15 Pro/Max or later, iPad with M-series chips or later)
- **On-Device vs Cloud:** AI processes can be used WITHOUT WiFi, indicating primarily on-device processing through Apple Intelligence's local models
- **Setup:** Users prompted to consent in iOS Settings first, then in app; initial setup takes a few minutes
- **Privacy:** Processing happens on-device; Apple Intelligence's privacy architecture applies

### AI Feature #1: AI Message Edit
- **Function:** Corrects grammar and spelling errors in messages composed in the Speech Display Bar (SDB)
- **Purpose:** Clear, confident communication
- **Additional:** Options for "creative AI" enhancements beyond basic correction
- **How to Access:** AI icon appears on right side of SDB; tap once for suggestions
- **Default State:** OFF -- user must explicitly toggle on

### AI Feature #2: AI Translate
- **Function:** Translates Speech Display Bar text into any language supported by Apple voices
- **Purpose:** Multi-lingual conversations, travel, bilingual settings
- **Voice Adaptation:** Voice output automatically adapts to target language
- **Location in App:** Found in "Modify Speech Display Bar" section of menu
- **Language Options:** Any language already in user's voice list
- **Default State:** OFF

### AI Feature #3: AI Image Generation
- **Function:** Creates new button/symbol images based on text prompts
- **How it Works:**
  1. Enter editing mode on a button
  2. Select "Generate Image" option
  3. Enter descriptive prompt and/or add reference picture
  4. Choose an art style
  5. AI generates up to 4 images at a time
- **Safety Guardrails:** Apple Intelligence guardrails prevent generation of images of people or copyrighted content
- **Purpose:** Create custom symbols when existing symbol libraries don't have what's needed
- **Default State:** OFF

### Clinical Significance
- AI features off by default -- clinicians/users choose when to enable each
- Especially benefits experienced AAC users and alternative access users
- Clinical use cases at many stages of AAC journey
- Represents first AI integration in PRC-Saltillo's 60-year history

Sources: [PRC-Saltillo AI Features Blog](https://prc-saltillo.com/blog/2026-ai-apps?mode=view), [2026.1 Update Blog](https://prc-saltillo.com/blog/app-update-2026-1)

## 2.5 Industry Context: AI/LLM in AAC

### ASHA Predictive Text Controversy (2025)
The American Speech-Language-Hearing Association (ASHA) hosted a "Future of AAC Research Summit" where predictive text generated significant debate:
- **Professionals' Concern:** Predictive text may compromise user autonomy and creative expression, potentially altering what users mean to say; technology rather than user controls conversation
- **AAC Users' Response:** Expressed worry about professionals becoming gatekeepers; noted they've adopted predictive text daily for emails, texts, social correspondence; voiced excitement about potential
- **Unresolved Issues:** Message authenticity concerns from communication partners, privacy/compensation for data, lack of clinician instruction on using predictive text

### Broader AAC AI Trends
- "The less I type, the better" -- research on AI language models enhancing AAC communication (ACM, 2023)
- Word prediction gaps: "Why Most AAC Word Predictions Miss the Mark" (NWACS, 2025)
- Consensus emerging that AI should augment, not replace, user agency in communication

Sources: [ASHA Predictive Text Article](https://leader.pubs.asha.org/do/10.1044/leader.FTR1.30032025.FAAC-predictive-text.36/full/), [ASHA Future of AAC](https://leader.pubs.asha.org/do/10.1044/leader.FTR1q.30092025.slp-aac-innovation.52/full/)

# TOPIC 3: EMOTIONAL TONE DETECTION & ADAPTATION

## 3.1 Avaz Expressive Tones -- Detailed Analysis

### What It Is
Avaz's Expressive Tones is a feature that infuses emotions into AAC text-to-speech output using AI-powered neural speech synthesis. It is exclusive to Avaz AAC and represents the most advanced emotional expression feature in any of PRC-Saltillo's portfolio apps.

### How It Works Technically
1. **Standard TTS Limitation:** Standard text-to-speech merely reads text with limited intonation, devoid of expressive tones
2. **Neural Enhancement:** Expressive Tones uses AI-powered speech synthesis to modify prosody (pitch, tempo, intonation, stress patterns) to convey specific emotions
3. **Cloud Processing:** Requires active internet connection -- processing happens on remote servers
4. **User-Selected (Manual):** User manually selects the desired emotional tone BEFORE the message is spoken -- it is NOT automatic emotion detection
5. **Emoji Interface:** Available tones represented by corresponding emojis for easy selection

### Supported Emotions/Tones (7 total)
1. **Anger** -- forceful, assertive intonation
2. **Excitement** -- enthusiastic, upbeat prosody
3. **Sadness** -- lower, slower delivery
4. **Curiosity** -- questioning, inquisitive tone
5. **Sarcasm** -- subtle tonal contradictions
6. **Whispering** -- quiet, intimate delivery
7. **Scared** -- anxious, trembling quality

### How to Enable
1. Ensure active internet connection
2. Navigate to Menu > Settings > Voice
3. Turn ON "Expressive Tones"
4. Select desired voice type
5. During communication, select tone before speaking message

### Limitations
- **English (US) only** as of current version
- **Requires internet** -- cannot be used offline
- **Manual selection only** -- no automatic emotion detection from text content or user behavior
- **No camera/facial recognition** input
- **No physiological data** integration

### Underlying Technology Assessment
- Likely uses cloud-based neural TTS service (possibly Amazon Polly, Google Cloud TTS, or proprietary)
- NOT using a general-purpose LLM -- this is specialized expressive speech synthesis
- Modifies prosodic features of synthesized speech rather than generating entirely new audio
- Cross-platform (iOS + Android) via cloud API

Sources: [Avaz Expressive Tones Blog](https://www.avazapp.com/blog/expressive-tones-on-avaz-aac/), [Avaz Expressive Tones Support](https://avazapp.freshdesk.com/support/solutions/articles/11000124901), [Avaz "Hear Me Out" Blog](https://avazapp.com/blog/expressive-tones-hear-me-out/)

## Critical Gaps Across All Three Apps

1. **No Automatic Emotion Detection:** None of the apps detect user emotions through any means (text analysis, camera, sensors, physiological data)
2. **No HealthKit/Health Data Integration:** Zero integration with health monitoring frameworks
3. **No Adaptive UI:** No app automatically adjusts interface complexity based on user state

5. **No AR/Camera Features:** Zero augmented reality or camera-based interaction
6. **Fragmented Ecosystem:** Post-acquisition (Avaz + Therapy Box), PRC-Saltillo runs 5+ separate app ecosystems with different data systems, support channels, and feature sets
7. **No Cross-Device Sync (PRC-Saltillo apps):** TouchChat and LAMP still require manual vocabulary transfer between devices
8. **Limited AI Integration:** Only Apple Intelligence (iOS26 only, specific hardware); no cross-platform AI; Avaz's cloud-based Expressive Tones is the only Android-compatible AI feature

2. **Health Data Integration:** HealthKit State of Mind API + wellness APIs are unused by any AAC app -- integration could enable adaptive interfaces
3. **Cross-Platform AI:** PRC-Saltillo's AI is iOS-only; Avaz's AI requires internet -- on-device, cross-platform AI for AAC is unaddressed
4. **Unified Data Platform:** Realize Language doesn't work with Avaz; no unified analytics across PRC-Saltillo's portfolio
5. **Fatigue-Adaptive Interfaces:** Grid simplification, break reminders, and performance monitoring based on usage patterns
6. **Camera-Based Context:** TrueDepth camera facial tracking + ARKit could enable context-aware communication support

1. **Deep Clinical Foundation:** 60 years of AAC expertise; LAMP's motor planning approach; WordPower's evidence base
2. **Professional Infrastructure:** Consultant network, AAC Language Lab, Learning Journey, funding support
3. **Hardware + Software Integration:** Only major AAC company with both dedicated devices and app ecosystem
4. **Avaz's Multilingual + Cross-Platform:** 15+ languages, iOS + Android, cloud backup
5. **Avaz's Expressive Tones:** Only emotional prosody feature in any PRC-Saltillo app -- unique differentiator
6. **Recent AI Investment:** Apple Intelligence integration shows willingness to adopt new technology
