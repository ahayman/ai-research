# Diagram Manifest: Writing System -- Simplified vs. Traditional vs. Alphabetic

## Diagrams

### Diagram 1: Six Methods of Character Simplification
- **File**: `diagram-1-six-simplification-methods.html`
- **Type**: Interactive card grid
- **Description**: Displays all six methods used in the 1956/1964 simplification scheme (cursive adoption, stroke reduction, component substitution, radical simplification, character merger, new creations) with before/after character examples, stroke reduction percentages, and structural consequences for each method.
- **Interactivity**: Click cards to highlight; hover examples for details.

### Diagram 2: Character Merger Ambiguity Map
- **File**: `diagram-2-character-merger-ambiguity-map.html`
- **Type**: Interactive expandable list
- **Description**: Shows Traditional-to-Simplified many-to-one mappings that create structural ambiguity. Seven major merger examples including 发(髮+發), 干(乾+幹+干), 台(臺+檯+颱+台), 后(後+后), 面(面+麵), 里(裏+里), 只(隻+只). Each expands to show disambiguation examples.
- **Interactivity**: Click merger rows to expand detail panels with contextual disambiguation examples.

### Diagram 3: Stroke Count Comparison
- **File**: `diagram-3-stroke-count-comparison.html`
- **Type**: Animated bar chart + comparison grid
- **Description**: Compares average strokes per character/letter across Traditional Chinese (12.2), Simplified Chinese (9.7), and English letters (2.5). Includes individual character comparisons (dragon, horse, country, side, big, gloomy) with percentage reduction badges.
- **Interactivity**: Animated bar fills on scroll; hover for details.

### Diagram 4: Three Writing System Architectures
- **File**: `diagram-4-writing-system-architecture.html`
- **Type**: Three-column comparison + flow diagrams + table
- **Description**: Comprehensive structural comparison of logographic (Traditional Chinese), reformed logographic (Simplified Chinese), and alphabetic (English) writing systems. Shows internal structure, phonetic information handling, reading paths (meaning-first vs. sound-first), and a detailed feature comparison table.
- **Interactivity**: Hover columns for highlight effect.

### Diagram 5: Simplification Timeline
- **File**: `diagram-5-simplification-timeline.html`
- **Type**: Vertical timeline with event cards
- **Description**: Chronological history from 1909 reform movement through 1956 scheme, 1964 General List, 1977 failed second round, 1986 rescission, 2000 language law, to 2013 current standard. Color-coded by event type (reform, success, failure, policy, context).
- **Interactivity**: Hover cards for highlight; color-coded dot system for event categorization.

### Diagram 6: Literacy Rate Impact Chart
- **File**: `diagram-6-literacy-rate-impact.html`
- **Type**: Dual-line chart with event markers + analysis cards
- **Description**: Plots literacy rates over time for mainland China (simplified) vs. Taiwan (traditional) from 1949-2020, with script reform events marked. Includes pro/con analysis cards and scholarly consensus verdict on whether simplification drove literacy improvements.
- **Interactivity**: Hover data points for exact values; event markers highlight reform dates.
