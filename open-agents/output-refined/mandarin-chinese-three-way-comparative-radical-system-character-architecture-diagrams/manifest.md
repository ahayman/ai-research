# Diagram Manifest: Radical System & Character Architecture

## Diagrams

1. **diagram-1-radical-frequency-pyramid.html** — Radical Frequency Pyramid
   - Tiered visualization of the 214 Kangxi radicals by frequency
   - Shows top 50 radicals with character counts
   - Animated cumulative coverage bar (23% → 43% → 58% → 78% → 100%)
   - Key stats: 214 total, 1,902 max (grass), 5 min (gen)

2. **diagram-2-semantic-field-clustering.html** — Semantic Field Clustering
   - 12 meaning domain clusters with color coding
   - Water, Fire, Wood, Earth, Metal, Body, Mind, Human, Animal, Movement, Sky, Shelter
   - Each cluster shows relevant radicals with example characters
   - Wuxing (Five Elements) connection at bottom

3. **diagram-3-positional-variant-gallery.html** — Positional Variant Gallery
   - 8 major radical transformations (水→氵, 人→亻, 心→忄, 手→扌, 火→灬, 刀→刂, 犬→犭, 示→礻)
   - Position tags (left, right, bottom)
   - Character examples with component breakdowns
   - Visual demo of uniform-square constraint

4. **diagram-4-semantic-phonetic-compound-anatomy.html** — Semantic-Phonetic Compound Anatomy
   - Central hub: 青 (qing) phonetic component
   - 6 family members (清/请/情/晴/精/睛) with breakdowns
   - Reliability statistics (80-90% compounds, 25-28% exact phonetic match, 65% semantic transparency)
   - Additional phonetic families (方, 包, 马, 且, 工, 巴)

5. **diagram-5-214-to-201-radical-transition.html** — 214→201 Radical Transition Map
   - Timeline from Zihui (1615) to GF 0011-2009 (2009)
   - System comparison (Kangxi 214 vs PRC 201+100)
   - Four change types: Removed, Simplified, Merged, Added
   - Practical impact assessment

6. **diagram-6-character-vs-morpheme-architecture.html** — Character Architecture vs Morpheme Architecture
   - Side-by-side comparison: Chinese 2D spatial vs English 1D linear
   - Visual composition demos (清 vs "uncomfortable")
   - Feature-by-feature comparison table (8 dimensions)
   - Same-concept comparisons (electricity, computer, telephone, psychology)

## Technical Notes
- All diagrams are standalone HTML with embedded CSS/JS
- Dark theme: --bg-primary: #0f172a; --accent: #06b6d4 (cyan)
- Responsive design with mobile breakpoints
- Animated progress bars on load
- Hover interactions on all interactive elements
- Chinese characters rendered with system fonts
