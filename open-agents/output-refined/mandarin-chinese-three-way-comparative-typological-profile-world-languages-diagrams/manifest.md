# Typological Profile -- Diagram Manifest

## Diagram Inventory

### 1. Morphological Typology Spectrum
- **File**: `diagram-1-morphological-spectrum.html`
- **Type**: SVG horizontal spectrum with language markers and verb form comparison bars
- **Content**: Isolating-to-polysynthetic scale with Mandarin (~1.2), English (~1.5), Turkish (~2.5), Arabic (~2.5-3.0), and Mohawk (~5-10+) positioned. Verb "eat" form counts: 1 (Mandarin), 5 (English), ~120 (Turkish), ~100+ (Arabic), 1000s (Mohawk). Cantonese shown clustering near Mandarin.
- **Research Section**: Section 2 (Morphological Typology)
- **Features**: Dark/light mode, Greenberg Index of Synthesis values, bar chart comparison

### 2. Word Order Parameter Grid
- **File**: `diagram-2-word-order-parameter-grid.html`
- **Type**: HTML table with color-coded cells and typological norm indicators
- **Content**: 10 word order parameters compared across Mandarin, English, and Cantonese. Color-coded: green = typical for SVO, red = rare for SVO, orange = OV-like feature. Highlights the relative clause anomaly (Mandarin prenominal RC in a VO language). Includes Chinese examples with pinyin.
- **Research Section**: Section 3 (Word Order Parameters)
- **Features**: Dark/light mode, Dryer (1992) citation, example sentences with glosses

### 3. Head-Directionality Mixed Profile
- **File**: `diagram-3-head-directionality-mixed.html`
- **Type**: HTML two-column layout showing head-initial vs head-final constructions
- **Content**: Mandarin's split personality: head-initial in VP (like English) vs head-final in NP (like Japanese). Three head-initial examples (VP, PP, complementizer) and four head-final examples (RC, genitive, adjective, aspect markers). Circumposition (在...上) shown as mixed. English (~90% head-initial), Mandarin (~50/50), Japanese (~95% head-final) comparison boxes.
- **Research Section**: Section 4 (Head-Directionality)
- **Features**: Dark/light mode, color-coded HEAD markers, three-language comparison

### 4. Tonal Typology World Map
- **File**: `diagram-4-tonal-typology-world.html`
- **Type**: HTML cards with geographic regions + horizontal bar chart of tone counts
- **Content**: Global distribution of tone languages across East/Southeast Asia, Sub-Saharan Africa, and Americas. Tone count comparison bars from 0 (English) through 2 (Igbo) to 14 (Chatino). Mandarin highlighted at 4 tones. Statistics: 40-70% of languages are tonal, 1.5B+ speakers.
- **Research Section**: Section 6 (Tonal Typology)
- **Features**: Dark/light mode, region cards with language examples, highlighted Mandarin row

### 5. Typological Scorecard
- **File**: `diagram-5-typological-scorecard.html`
- **Type**: HTML table with 18 parameters across Mandarin, English, and Cantonese
- **Content**: Comprehensive parameter comparison covering phonology (3), morphology (5), syntax (7), semantics (2), and writing (1). SAME/DIFF tags for Mandarin-English and Mandarin-Cantonese. Summary: CMN=ENG on 3/18 (17%), CMN=YUE on 13/18 (72%).
- **Research Section**: Section 9 (WALS-Style Feature Summary)
- **Features**: Dark/light mode, section grouping, summary statistics

### 6. WALS-Style Feature Rarity Map
- **File**: `diagram-6-wals-feature-comparison.html`
- **Type**: HTML horizontal bar chart with rarity classifications
- **Content**: 14 typological features with percentage of world languages sharing Mandarin's value. Three tiers: Common (>40%: SVO, prepositions, Adj-N, Dem-N), Minority (15-40%: isolating, tone, topic-prominent, pro-drop, classifiers, no gender, SVCs), Rare (<10%: SVO+prenominal RC at <2%, mixed head-directionality at ~5%, zero inflection at ~8%).
- **Research Section**: Section 11 (Typological Uniqueness Score)
- **Features**: Dark/light mode, percentage bars, rarity tags, three-tier classification
