## Diagram Descriptions

### Diagram 1: First Language Acquisition Timeline
- **Type**: SVG horizontal timeline
- **Elements**: 6 stage boxes along a timeline axis, vocabulary badges, pattern summary, growth bar
- **Key Insight**: Children around the world reach language milestones in the same invariant sequence: sounds, syllables, words, word combinations, grammar, complex grammar. This universality suggests a biologically constrained developmental program. Vocabulary grows from 0 to 10,000+ words across the stages.
- **Accessibility**: Six boxes arranged horizontally along a timeline from birth to 5+ years. Stage 1 Prelinguistic (0-12mo): cooing, babbling, sound discrimination, 0 words. Stage 2 One-Word (12-18mo): first words, holophrastic speech, ~50 words. Stage 3 Two-Word (18-24mo): vocabulary explosion, combinations like "more milk," rudimentary syntax, 200-300 words. Stage 4 Telegraphic (24-30mo): multi-word utterances missing function words, overgeneralization errors proving rule extraction, 500-1000 words. Stage 5 Complex Grammar (30-60mo): full sentences with embedded clauses, questions and negation, 2000-5000+ words. Stage 6 Mastery (5+ years): literacy, academic language, metalinguistic awareness, 10,000+ words.

### Diagram 2: SLA Theory Comparison
- **Type**: Mermaid flowchart (three-column comparison)
- **Elements**: 3 theory subgraphs with 5 nodes each, 1 synthesis subgraph with circular flow
- **Key Insight**: Krashen's Input Hypothesis, Long's Interaction Hypothesis, and Swain's Output Hypothesis are complementary, not competing. Input provides raw material, interaction makes input comprehensible and provides feedback, output forces deep grammatical processing and reveals knowledge gaps. The synthesis shows them as a cycle: Input feeds Interaction, which enables Output, which reveals gaps triggering need for more Input.
- **Accessibility**: Three side-by-side columns. Krashen: core claim (comprehensible input i+1), acquisition vs. learning distinction, the Monitor, affective filter, practical rule (aim 70-90% comprehension). Long: core claim (interaction makes input comprehensible), negotiation of meaning, modified input, implicit correction through recasts, practical rule (conversation provides input AND feedback). Swain: core claim (output forces deeper processing), noticing function (reveals gaps), hypothesis testing, metalinguistic function (syntactic processing), practical rule (understanding input semantically does not require grammar; output demands it). Synthesis: Input feeds Interaction, Interaction enables Output, Output gaps trigger need for more Input -- a virtuous cycle.

### Diagram 3: Cross-Linguistic Transfer Map
- **Type**: Mermaid concept map (flowchart)
- **Elements**: Central node, 2 main branches (positive/negative), 5 examples each, predictor subgraph, strategy node
- **Key Insight**: Transfer from known languages is predictable from structural comparison. Positive transfer (shared word order, cognates, grammatical concepts) accelerates learning. Negative transfer (missing categories, different word order, false friends) causes interference. Typological distance determines the ratio. Close languages (English-Dutch) yield mostly positive transfer; distant languages (English-Japanese) yield mostly negative transfer risk.
- **Accessibility**: Central node "Your Known Languages" branches to three areas. Positive Transfer: shared word order (English SVO to French SVO), cognate vocabulary (university/universidad), shared grammatical concepts (articles), similar morphology (Spanish to Italian), case system familiarity (German to Russian). Negative Transfer: missing category (no gender in English to French), different word order (SVO to SOV/V2), missing morphology (Mandarin no conjugation to Spanish), false friends (embarrassed vs embarazada), structural mismatch (prepositions vs postpositions). Typological Distance predictor: close (same family, max positive), medium (same family different features, mixed), far (different family, max negative risk).

### Diagram 4: FSI Language Difficulty Rankings
- **Type**: SVG tiered pyramid
- **Elements**: 4 pyramid tiers, difficulty factors panel, pattern summary box, time comparison bars
- **Key Insight**: The FSI rankings measure typological distance from English, not absolute complexity. Category I languages (Spanish, French) take 600-750 hours because they share vocabulary, grammar, and script with English. Category IV languages (Arabic, Chinese, Japanese, Korean) take 2,200 hours because they differ in script, vocabulary, grammar, word order, morphology, phonology, and social systems.
- **Accessibility**: Pyramid with four tiers from easiest (top) to hardest (bottom). Category I (24-30 weeks, 600-750 hours): Spanish, French, Italian, Portuguese, Dutch, Norwegian, Swedish, Danish, Romanian. Category II (36 weeks, 900 hours): German, Indonesian, Malay, Swahili -- similar roots but added complexity. Category III (44 weeks, 1100 hours): Russian, Greek, Hindi, Hebrew, Polish, Czech, Finnish, Hungarian, Turkish, Thai, Vietnamese, Persian, Urdu, Tagalog -- significant structural differences. Category IV "Super-Hard" (88 weeks, 2200 hours): Arabic, Mandarin, Cantonese, Japanese, Korean -- different script, no cognates, fundamentally different grammar, tones/honorifics. Side panel lists seven difficulty factors: script, vocabulary overlap, grammar, word order, morphology, phonology, social systems.

### Diagram 5: The Pattern Learner's Structural Toolkit
- **Type**: Mermaid decision tree (flowchart)
- **Elements**: 1 start node, 3 question nodes, 3 subgraph branches with 4+3+4 type nodes and matching strategy nodes
- **Key Insight**: Three diagnostic questions -- morphological type, word order, and case marking -- create a systematic framework for approaching any new language. Each answer leads to specific, actionable learning strategies. This toolkit synthesizes typological knowledge from the entire deep dive into a practical tool particularly suited to pattern-oriented and systemizing cognitive styles.
- **Accessibility**: Start node "Encountering a New Language" leads to three parallel questions. Question 1 (Morphological Type): Isolating (Mandarin -- focus on word order, particles, tones), Agglutinative (Turkish -- master affix inventory, one meaning per affix, build like LEGO), Fusional (Russian -- learn paradigm tables, expect irregularities), Polysynthetic (Inuktitut -- one word equals whole clause). Question 2 (Word Order): SOV (Japanese -- postpositions, genitive before noun, head-final), SVO (English -- prepositions, mixed, variable), VSO (Arabic -- always prepositions, genitive after noun, head-initial). Question 3 (Case Marking): No case (English -- word order is sacred), Nominative-Accusative (German -- learn case paradigms, gain word order freedom), Ergative-Absolutive (Basque -- reframe subject concept), Split system (Hindi -- track which contexts trigger which alignment).

### Diagram 6: The Multilingual Mind
- **Type**: SVG layered diagram
- **Elements**: 4 concentric layers (executive control, phonological, lexical, conceptual), 3 language columns, cross-activation arrows, cumulative advantage box
- **Key Insight**: The multilingual lexicon is integrated but differentiated. All languages share a common conceptual system (the meanings), but maintain separate phonological and lexical systems. These systems are always co-activated -- hearing a word in one language automatically activates its translation equivalent. Executive control manages which language is dominant. The system is additive: each new language expands the network.
- **Accessibility**: Outermost layer: Executive Control System (selectively activates/inhibits languages). Second layer: three phonological systems side by side (L1 English, L2 Spanish, L3 Japanese) with cross-activation arrows between them. Third layer: three lexical systems (separate lemma entries for each language) with cross-activation arrows. Innermost layer: Shared Conceptual System where "dog" = "perro" = "inu" all point to the same concept. Vertical arrows connect each language's lexicon down to the shared concepts. Bottom box describes the cumulative advantage: each language connects to the existing network, the brain creates parallel systems rather than filling up.

### Diagram 7: Explicit-to-Implicit Knowledge Pipeline
- **Type**: Mermaid flowchart (linear pipeline with feedback)
- **Elements**: 5 main pipeline stages, 1 feedback loop subgraph, 1 interface theory subgraph
- **Key Insight**: Based on Nick Ellis's weak interface hypothesis, explicit structural knowledge (grammar rules, typological awareness) facilitates implicit learning by enabling "noticing." The pipeline runs: Explicit Knowledge enables Noticing, which guides attention during Practice, which leads to Proceduralization (faster access), which becomes Implicit Knowledge (automatic, intuitive). Feedback loops ensure that output errors are diagnosed and fed back into explicit analysis.
- **Accessibility**: Five sequential stages connected left to right. Stage 1 Explicit Knowledge: grammar rules, paradigm tables, typological classification, sourced from textbooks and this deep dive. Stage 2 Noticing: explicit knowledge directs attention to specific features in input (e.g., "that suffix is past tense"). Stage 3 Practice: repeated exposure and use through comprehensible input, interaction, and output production; hundreds to thousands of encounters needed. Stage 4 Proceduralization: knowledge becomes faster to access, less hesitation, less mental effort. Stage 5 Implicit Knowledge: fluent, automatic, intuitive performance. Feedback loop: output errors and gaps discovered during practice feed back to explicit analysis (Is this L1 transfer? Overgeneralization? Paradigm gap?) which re-enters the cycle. Bottom box explains the weak interface position: explicit and implicit are different systems, but explicit facilitates implicit through noticing.
