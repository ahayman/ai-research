# English Syntax -- Diagram Manifest

### 1. Basic SVO Sentence Structure
- **File**: `diagram-1-basic-svo-structure.html`

### 2. Phrase Structure (NP, VP, PP Hierarchy)
- **File**: `diagram-2-phrase-structure.html`
- **Type**: Phrase type cards + head-initial note + recursive embedding SVG
- **Content**: Five phrase types (NP, VP, PP, AdjP, AdvP) each with template, head identification, and examples. Head-initial principle with examples and Germanic adjective exception noted. SVG showing four levels of recursive phrase embedding in "The tall student quickly read the interesting book in the old library."
- **Research Section**: Component 2 (Phrase Structure)
- **Features**: Dark/light mode, color-coded phrase types, nesting visualization

### 3. Question Formation: Do-Support and Inversion
- **File**: `diagram-3-question-formation.html`
- **Type**: Flowchart + transformation examples + cross-linguistic comparison
- **Content**: Decision flowchart for question formation (auxiliary present? invert : insert DO + invert). Step-by-step wh-question derivation showing wh-movement and do-support. Negation and emphasis examples. Cross-linguistic comparison table (English, German, French, Japanese). Historical note on loss of verb-second and do-support as structural patch.
- **Research Section**: Component 5 (Auxiliary System and Do-Support), Component 6 (SAI)
- **Validation**: Do-support typological uniqueness confirmed (validation claim 3)
- **Features**: Dark/light mode, branching flowchart, cross-linguistic data

### 4. OSASCOMP Adjective Order
- **File**: `diagram-4-osascomp-adjective-order.html`
- **Type**: Colored slot sequence + gradient bar + full example + violation demo
- **Content**: Eight OSASCOMP slots (Opinion-Size-Age-Shape-Color-Origin-Material-Purpose) each color-coded with example adjectives. Subjective-to-objective gradient visualization. Full example "a lovely little old rectangular green French silver whittling knife" with each adjective color-matched to its slot. Violation demonstration showing reordered adjectives.
- **Research Section**: Component 4 (OSASCOMP Adjective Order)
- **Validation**: Described as "strong preference" per validator (claim 2)
- **Features**: Dark/light mode, color-coded slots, gradient principle visualization

### 5. Relative Clause Types
- **File**: `diagram-5-relative-clause-types.html`
- **Type**: Side-by-side comparison + relativizer selection table
- **Content**: Restrictive vs. non-restrictive relative clauses with examples, rules, and comma patterns. Relativizer selection guide (who, whom, which, that, zero, whose, where/when) showing which are allowed in each clause type. The that/which formal writing rule explained.

- **Features**: Dark/light mode, comparison layout, comprehensive selection table
