## Diagram Descriptions

### Diagram 1: The Five Competing Models — Landscape Map
- **Type**: Architecture/landscape diagram
- **Elements**: 5 model cards + interoception cluster + central PVT card (in contested color)
- **Key Insight**: PVT exists in a crowded landscape; five mainstream alternatives explain the same data
- **Accessibility**: All elements have text labels and descriptions; color is supplementary

### Diagram 2: NVI Mechanism — vmPFC → Amygdala → Vagus
- **Type**: SVG flow diagram with annotations
- **Elements**: 4-layer vertical flow (vmPFC → Amygdala → Brainstem → Heart/HRV), with annotation boxes for predictions, exclusions, and evidence base
- **Key Insight**: NVI predicts the same HRV findings as PVT without phylogenetic claims; has higher academic citation impact

### Diagram 3: Predictive Processing — Emotion as Interoceptive Inference
- **Type**: SVG cyclic diagram
- **Elements**: Brain (predictions, top), body (signals, bottom), active inference loop, emotion construction synthesis at base
- **Key Insight**: PP absorbs neuroception as ordinary predictive inference; explains autism + alexithymia in a way PVT cannot

### Diagram 4: Allostatic Load Curve Applied to Autistic Burnout
- **Type**: SVG line graph
- **Elements**: Time axis, allostatic load axis, rising load curve with annotated autistic stressors, inflection at burnout, recovery dashed line
- **Key Insight**: Burnout is allostatic-load shaped, not "dorsal vagal" shaped; recovery requires demand reduction

### Diagram 5: Constructed Emotion vs. Hardwired Emotion Programs
- **Type**: Side-by-side comparison panels
- **Elements**: PVT panel (red border, hardwired states) and Barrett panel (green border, constructed emotion); formula, predictions, evidence
- **Key Insight**: Lindquist 2012 meta-analysis supports constructed emotion; PVT's discrete-state architecture is directly contradicted

### Diagram 6: Somatic Markers — Damasio's Body-Guided Decision
- **Type**: SVG decision-flow diagram
- **Elements**: Decision point → body response → positive/negative markers → vmPFC integration → biased decision
- **Key Insight**: Damasio's framework captures the "body feels it first" insight without PVT's contested phylogenetic claims

### Diagram 7: The Convergence/Divergence Map
- **Type**: 5-row × 7-column matrix with color-coded cells
- **Elements**: 5 phenomena × 7 models; color codes for fit quality; verdict row at bottom
- **Key Insight**: PVT has no unique explanatory advantage on any phenomenon that matters; allostasis dominates for burnout; NVI/PP dominate for HRV
- **This is the chapter's central analytical anchor**

### Diagram 8: "Is This the Right Framework?" — Decision Aid
- **Type**: Branching decision-aid grid
- **Elements**: 8 question types each with framework recommendation or rejection
- **Key Insight**: PVT useful as clinical heuristic, not as mechanism; allostasis for burnout; NVI for HRV; PP for emotion construction

### Diagram 9: The Interoception Stack — Craig, Garfinkel, Bird & Cook
- **Type**: Layered hierarchy (bottom-up)
- **Elements**: 3 layers (insular substrate, three-dimensional model, alexithymia hypothesis) with citations and findings
- **Key Insight**: Interoception research has fine-grained mechanisms PVT cannot represent

### Diagram 10: Pellicano & Burr — Hypo-Priors Model of Autism
- **Type**: SVG Bayesian comparison
- **Elements**: Normal Bayesian inference (strong prior) vs. autistic hypo-priors (weak prior, dominant sensory data); consequences for each
- **Key Insight**: Sensory hypersensitivity has a predictive-coding explanation that's more testable than PVT's "defensive state"
