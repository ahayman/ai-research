# Diagram Manifest: Digital-Age Mandarin — Internet Language, Tech Input & Linguistic Innovation

## Category
Mandarin Chinese Three-Way Comparative Deep Dive — Category 15

## Diagrams

### Diagram 1: Pinyin IME Input Flow
- **File**: `diagram-1-pinyin-ime-input-flow.html`
- **Type**: Process flow with comparison
- **Description**: Shows the two-stage Pinyin input process (phonological encoding → character selection), the skills required vs. not required, and a three-way comparison of input method effects across PRC, Taiwan, and English contexts.
- **Iframe height**: 700px

### Diagram 2: Internet Slang Taxonomy
- **File**: `diagram-2-internet-slang-taxonomy.html`
- **Type**: Classification grid (9 categories)
- **Description**: Comprehensive taxonomy of Chinese internet word-formation patterns: number homophones, Pinyin abbreviations, homophone substitution, character decomposition, semantic extension, alphanumeric code-mixing, emoji pragmatics, censorship-driven coinage, and platform-specific jargon. Each with examples and English comparison notes.
- **Iframe height**: 850px

### Diagram 3: Censorship-Creativity Cycle
- **File**: `diagram-3-censorship-creativity-cycle.html`
- **Type**: Cyclical process diagram with case study
- **Description**: The five-step cycle (block → create substitute → spread → detect → block substitute → repeat) with a detailed case study of 和谐/河蟹, plus four evasion strategy categories.
- **Iframe height**: 800px

### Diagram 4: Platform Register Spectrum
- **File**: `diagram-4-platform-register-spectrum.html`
- **Type**: Spectrum / comparative cards
- **Description**: Six Chinese social media platforms (Government Weibo, Weibo, WeChat, Douyin, Bilibili, Xiaohongshu) arranged along a formality spectrum, with register descriptions, slang levels, feature tags, and sample language.
- **Iframe height**: 950px

### Diagram 5: Character Amnesia — Causes and Effects
- **File**: `diagram-5-character-amnesia-causes-effects.html`
- **Type**: Cause-effect flow diagram
- **Description**: Traces the path from Pinyin input adoption through the recognition-recall dissociation (提笔忘字) to cultural and educational effects. Includes preserved vs. degraded skill comparison, statistics, and government response measures.
- **Iframe height**: 900px

### Diagram 6: Code-Mixing Patterns
- **File**: `diagram-6-code-mixing-patterns.html`
- **Type**: Pattern gallery with annotated examples
- **Description**: Six code-mixing patterns (English verb + Chinese grammar, Pinyin abbreviations, number encoding, English root + Chinese aspect, letter-character compounds, embedded English phrases) with color-coded sentence examples showing how Chinese grammar productively absorbs English material.
- **Iframe height**: 900px
