## Diagram Descriptions

### Diagram 1: Root-Pattern Multiplication Table
- **Type**: Matrix/grid with computed cells
- **Elements**: 5 roots (k-t-b, '-l-m, d-r-s, s-l-m, q-r-') x 8 patterns (Form I verb, Form II verb, active participle, passive participle, noun of place, masdar, noun of instrument, adjective). Each cell = derived word. Learning items counter: 13 items learned -> 40 words produced. English comparison: 40 separate memorization items required.
- **Key Insight**: Adding ONE new root immediately generates 8+ new words. Adding ONE new pattern multiplies with every known root. This is EXPONENTIAL, not linear.
- **English Comparison**: English vocabulary is a museum (individual items). Arabic vocabulary is a factory (generative system).

### Diagram 2: Root Extraction Decision Tree
- **Type**: Step-by-step flowchart
- **Elements**: Four-step algorithm: strip prefixes (al-, m-, t-, ist-, in-) -> strip suffixes (ta' marbuta, -at, -un/-in) -> identify 3 core consonants -> verify against known root. Example paths traced: maktaba -> k-t-b, mu'allim -> '-l-m.
- **Key Insight**: This algorithm works on THOUSANDS of words. It is a genuine cognitive tool, not a pedagogical gimmick.
- **English Comparison**: English has no equivalent algorithm. "Library" doesn't visibly connect to "book."

### Diagram 3: Verb Form Identification Flowchart (I-X)
- **Type**: Branching flowchart with semantic function labels
- **Elements**: Entry: "Given an Arabic verb, identify its form." First branch: prefix detection (ta- -> V/VI, ista- -> X, in- -> VII, a- -> IV). Second branch: internal features (doubled middle -> II, long a -> III, infix -t- -> VIII). Each terminal: form number + semantic function + example.
- **Key Insight**: Like a periodic table of semantic modifications -- once you know the table, you can predict properties of any element.
- **English Comparison**: English uses separate verbs, phrasal verbs, and auxiliary constructions with no consistent pattern.

### Diagram 4: Broken Plural Pattern Map
- **Type**: Pattern matching chart with transformation arrows
- **Elements**: 8 rows showing singular pattern -> plural pattern -> 3 examples each. Visual template showing which letters stay/change/are added. Frequency indicators. The 80/20 insight: 8 patterns cover ~70% of cases.
- **Key Insight**: Broken plurals are PATTERNED, not random. There are rules within the "irregularity."
- **English Comparison**: English irregular plurals (~20 words) are truly arbitrary. Arabic broken plurals (~30 patterns) are systematic.

### Diagram 5: Gender Prediction Decision Tree
- **Type**: Simple branching decision tree
- **Elements**: Primary rule: ta' marbuta -> FEMININE (~85% accuracy). Secondary markers: alif maqsura, alif mamduda. Semantic feminines: paired body parts, natural forces. Exception list (< 20 common masculine words ending in ta' marbuta). Non-human plural rule: always feminine singular agreement.
- **Key Insight**: Arabic gender is MUCH more predictable than French or German -- the ta' marbuta rule alone covers the vast majority.
- **English Comparison**: English has no grammatical gender. French gender is largely arbitrary. Arabic gender is mostly predictable from word shape.

### Diagram 6: Arabic Regularity vs. English Irregularity
- **Type**: Comparative scorecard / side-by-side evaluation
- **Elements**: 7 dimensions: verb conjugation regularity, plural formation, spelling-pronunciation correspondence, vocabulary transparency, gender predictability, case marking clarity, learning curve shape. Arabic and English scored on each. Summary: Arabic = systematically complex; English = randomly complex.
- **Key Insight**: For pattern learners, systematic complexity is tractable. Random complexity is not.
- **Validation Note**: FSI Category V (corrected from IV), 2,200 hours.

### Diagram 7: Optimal Learning Sequence for Pattern Learners
- **Type**: Six-phase roadmap with dependencies
- **Elements**: Phase 1: Script (weeks 1-3). Phase 2: Root System (weeks 3-8). Phase 3: Verb Forms I-X (weeks 6-12). Phase 4: Noun Patterns (weeks 8-16). Phase 5: Grammar Skeleton (weeks 12-20). Phase 6: Dialect Layer (weeks 16+). Each phase builds on previous. Pattern-system growth indicator at each step.
- **Key Insight**: System first, variations later -- the pattern learner's optimal sequence.
- **English Comparison**: No equivalent systematic learning sequence exists for English because English has no equivalent system to learn.
