# Diagram Manifest: Semantics & Compound Formation
## Mandarin Chinese Three-Way Comparative

### Diagrams

1. **diagram-1-five-compound-types.html**
   - Five-column grid showing each compound type with structural formula, examples, and productivity rating
   - Types: Coordinate, Modifier-Head, Verb-Object, Verb-Complement, Subject-Predicate
   - Each example shows Chinese characters, pinyin, component meanings, and compound meaning
   - Summary bar with key statistics (80% compounds, 95% neologisms, 2-syllable norm)

2. **diagram-2-morpheme-network.html**
   - Three morpheme pivot hubs: 电 (electricity), 学 (study), 机 (machine)
   - Each hub shows compounds where the morpheme appears as modifier and as head
   - Demonstrates how ~1,500 morphemes unlock 30,000+ compounds
   - Insight box explaining combinatorial vocabulary growth

3. **diagram-3-semantic-transparency-spectrum.html**
   - Gradient from transparent to opaque with three labeled zones
   - Transparent examples: 电话, 火山, 飞机, 洗衣机, 大学
   - Semi-transparent examples with explanatory notes: 小心, 马路, 吃醋, 小说, 风流
   - Opaque examples: 东西, 马虎, 尴尬, 蝴蝶
   - Cross-linguistic transparency comparison (Mandarin vs English vs Classical)

4. **diagram-4-vocabulary-architecture-comparison.html**
   - Three-column comparison: Modern Mandarin, English, Classical Chinese
   - Each column shows word-formation strategies with percentage bars
   - Word length comparison with visual blocks
   - Neologism example: how each system creates "computer"

5. **diagram-5-loanword-integration-strategies.html**
   - Four strategies: Semantic calque, Phonetic transliteration, Hybrid, Sound+meaning matched
   - Each strategy with frequency, description, and multiple examples
   - English open-door borrowing contrast panel
   - Four reasons why Mandarin prefers calques

### Design Notes
- Dark theme: --bg: #0f172a, --accent: #06b6d4 (consistent with master deep dive theme)
- Light/dark mode via prefers-color-scheme media query
- All diagrams are standalone HTML files with no external dependencies
- Responsive grid layouts with mobile breakpoints
- Chinese characters displayed at readable sizes with pinyin and literal translations
