## Diagram Descriptions

### Diagram 1: The Four-Space Model
- **Type**: Custom SVG
- **Elements**: 4 labeled spaces with contents, cross-space mapping lines, selective projection arrows
- **Key Insight**: Every conceptual integration network has at minimum four mental spaces: Input 1, Input 2, Generic Space (shared abstract structure), and Blended Space (where emergent meaning arises). The surgeon-butcher example demonstrates how "incompetence" is not a property of either butchers or surgeons but emerges from the blend.
- **Accessibility**: Four rectangles arranged in a diamond pattern. Top center: Generic Space containing "Agent, undergoer, sharp instrument, cutting action, workspace, goal." Left middle: Input Space 1 (Surgery) containing surgeon, patient, scalpel, operating room, goal of healing. Right middle: Input Space 2 (Butchery) containing butcher, animal carcass, cleaver, abattoir, goal of cutting meat. Bottom center: Blended Space containing the surgeon with the butcher's methods, emergent meaning "incompetence." Dashed purple lines connect generic space to both inputs. Dashed red lines show cross-space mappings between counterpart elements. Blue and amber arrows show selective projection into the blend.

### Diagram 2: Emergent Structure Process
- **Type**: Mermaid Flowchart
- **Elements**: 3 process stages, 3 examples, 1 emergent meaning node, 2 feedback arrows
- **Key Insight**: Blends generate meaning through three sequential-but-recursive processes: Composition (juxtaposing elements from different inputs), Completion (recruiting background knowledge), and Elaboration (running the blend as mental simulation). The process is recursive: elaboration triggers further completion, and emergent structure feeds into further blending.
- **Accessibility**: Vertical flowchart. Input Spaces Activated flows to Composition (juxtaposing elements), which flows to Completion (background knowledge fills in), which flows to Elaboration (running the blend). Each stage has an adjacent example box showing the surgeon-butcher case. Elaboration flows to Emergent Meaning (incompetence). Dashed feedback arrow from Elaboration back to Completion. Dashed recursion arrow from Emergent Meaning back to Input Spaces.

### Diagram 3: Vital Relations Map
- **Type**: Mermaid Mind Map
- **Elements**: 1 root node, 6 cluster branches, 15 vital relation nodes, descriptions and compression examples
- **Key Insight**: Fauconnier and Turner identify 15 fundamental vital relations organized into 6 functional clusters. Blending compresses outer-space vital relations (between mental spaces) into inner-space relations (within the blend) to achieve "human scale" -- making complex concepts cognitively manageable.
- **Accessibility**: Mind map with central node "VITAL RELATIONS." Six branches radiate outward. Spatiotemporal: Time, Space. Causal: Cause-Effect, Intentionality, Change. Identity: Identity, Uniqueness, Role-Value. Structural: Part-Whole, Category, Property. Comparative: Analogy, Disanalogy, Similarity. Semiotic: Representation. Each relation has a description and compression example.

### Diagram 4: Example Blends Compared
- **Type**: Custom SVG (3 side-by-side four-space diagrams)
- **Elements**: 3 blend cards, each with Generic Space, Input 1, Input 2, Blend, and emergent structure label
- **Key Insight**: The same four-space architecture produces radically different emergent meanings depending on the network type. Single-scope: one frame dominates (incompetence). Mirror: shared frame, different content (meeting point). Double-scope: both frames contribute (death at human scale).
- **Accessibility**: Three cards arranged horizontally. Left card (blue border, "Single-Scope"): surgeon-butcher example with surgery frame dominating and butchery providing content; emergent meaning is incompetence. Center card (purple border, "Mirror"): Buddhist Monk riddle with shared walking-on-path frame; two days compressed into one, producing the emergent meeting point. Right card (red border, "Double-Scope"): Grim Reaper with both Death (causation frame) and Human Agent (agency frame) contributing; emergent meaning is death at human scale.

### Diagram 5: CMT vs. Blending Theory Comparison
- **Type**: Mixed SVG + Mermaid
- **Elements**: 2 comparison panels (CMT and Blending), feature lists, network type spectrum
- **Key Insight**: CMT and Blending Theory are complementary. CMT describes stable metaphorical mappings (Source to Target, unidirectional) and maps onto just one network type: single-scope. Blending Theory also handles simplex, mirror, and double-scope networks. The spectrum from simplex to double-scope represents increasing cognitive complexity.
- **Accessibility**: Two side-by-side panels. Left panel (CMT): SVG showing Source domain with unidirectional arrow to Target domain. Feature list: two domains, unidirectional, stable/entrenched, diachronic, automatic, no emergent structure focus. Right panel (Blending): SVG showing four-space diamond with Generic Space, Input 1, Input 2, and Blended Space with emergent structure. Feature list: four+ spaces, multi-directional, dynamic/on-line, synchronic, effortful, emergent meaning central. Below: horizontal spectrum showing Simplex, Mirror, Single-Scope (highlighted as where CMT maps), Double-Scope, with increasing complexity.

### Diagram 6: Construction Grammar as Blending
- **Type**: Mermaid Flowchart
- **Elements**: Generic space, 2 inputs (lexical frame and constructional frame), blend, emergent meaning, principle statement
- **Key Insight**: Grammatical constructions are form-meaning pairings that blend with lexical content. In "He sneezed the napkin off the table," the intransitive verb "sneeze" is coerced by the caused-motion construction into a causal interpretation. The emergent meaning -- involuntary action interpreted as causation -- exists in neither the verb frame nor the construction alone.
- **Accessibility**: Vertical flowchart. Generic Space (agent performs action, entity changes state) connects to Input 1 (SNEEZE frame: intransitive, involuntary, no object) and Input 2 (CAUSED-MOTION construction: X causes Y to move to Z, transitive, volitional). Both inputs project into Blended Space (sneeze-caused-motion event). Blend flows to Emergent Meaning (involuntary action interpreted as volitional causation). Emergent Meaning flows to Principle (construction meaning is independent of specific words).

### Diagram 7: Blending Recursion
- **Type**: Mermaid Flowchart
- **Elements**: 4 levels of recursive blending, 8 input/output nodes, connecting arrows, summary pattern
- **Key Insight**: Blending is recursive -- each blend's output can become input to the next blend. This allows unbounded conceptual complexity from finite cognitive resources. The example traces four levels: harvest metaphor to Time-as-Reaper to Grim Reaper to chess-with-Death to Bergman's Seventh Seal. Fauconnier and Turner hypothesize that this recursive compositionality is uniquely human.
- **Accessibility**: Four grouped levels arranged vertically. Level 1: Input A (harvest/reaping) + Input B (passage of time) produce Blend 1 (Time as Reaper). Level 2: Blend 1 reused as input + Input C (human agent) produce Blend 2 (Grim Reaper). Level 3: Blend 2 reused as input + Input D (chess game) produce Blend 3 (playing chess with Death). Level 4: Blend 3 reused as input + Input E (film narrative) produce Blend 4 (Bergman's Seventh Seal). Dashed arrows show each blend becoming input to the next level. Final node states the recursive pattern principle.
