# Chinese as an Isolating Language -- Diagram Manifest

## Diagram Inventory

### 1. The Isolating-Synthetic Spectrum -- Where Languages Fall
- **File**: `diagram-1-isolating-synthetic-spectrum.html`
- **Type**: SVG spectrum diagram with language boxes
- **Content**: Gradient spectrum from isolating to synthetic with four language markers: Chinese (far left), English (left-center), French (center-right), Arabic (right). Each language has a detail box showing how "I went" is expressed, verb form count, and grammar mechanism. Bottom comparison table shows how each language handles tense, plural, gender, and questions.
- **Research Section**: Overview + Component 4 (Missing Grammar)
- **Features**: Dark/light mode, four-language comparison, inflection gradient

### 2. The Three De Particles -- Function Mapping
- **File**: `diagram-2-three-de-particles.html`
- **Type**: SVG triptych pattern diagram
- **Content**: Three sections for 的 (attributive/possessive modifying nouns), 地 (adverbial modifying verbs), and 得 (complement evaluating verbs). Each section shows the pattern formula, 2-3 Chinese examples, and English equivalents. Bottom box shows the elegant triptych pattern. English comparison shows how 的 replaces possessives, adjective positioning, and relative clauses.
- **Research Section**: Component 1 (The Structural Particles)
- **Features**: Dark/light mode, color-coded sections, pattern formulas

### 3. Aspect vs. Tense -- Chinese vs. English Time Systems
- **File**: `diagram-3-aspect-vs-tense.html`
- **Type**: HTML side-by-side comparison + mapping table
- **Content**: English 12+ verb form grid (past/present/future x simple/progressive/perfect) alongside Chinese 3-particle system (了/着/过). Critical warning about 了 not meaning "past tense." Mapping table showing how English forms correspond to Chinese constructions. Insight box on optional vs. obligatory time marking.
- **Research Section**: Component 2 (The Aspect Particles)
- **Features**: Dark/light mode, tense-aspect grid, misconception warning

### 4. What Chinese Does NOT Have -- The Absence Inventory
- **File**: `diagram-4-missing-grammar.html`
- **Type**: HTML comparison matrix
- **Content**: Six-row matrix comparing verb conjugation, noun plurals, grammatical gender, articles, case marking, and tense marking across Chinese, English, French, and Arabic. Each Chinese "NO" cell linked to the replacement mechanism (word order, particles, context, classifiers). Insight box on rearrangement rather than absence.
- **Research Section**: Component 4 (What Chinese Does NOT Have)
- **Features**: Dark/light mode, four-language gradient, replacement mechanisms

### 5. Chinese Particle System Architecture -- Complete Map
- **File**: `diagram-5-particle-system-map.html`
- **Type**: SVG mind map / classification tree
- **Content**: Central hub branching to four categories: structural (的/地/得), aspect (了/着/过), modal (吗/吧/呢/啊), and other functional words (把/被/在 + modal verbs). Each particle has function label and example. Cross-link note for 了 appearing in both aspect and sentence-final. Insight box: ~15-20 particles handle most of Chinese grammar.
- **Research Section**: All Components (System Overview)
- **Features**: Dark/light mode, four-branch classification, finite system insight
