The Hobson 2019 Multiroute Model

Hobson, Brewer, Catmur & Bird (2019). The Role of Language in Alexithymia: Moving Towards a Multiroute Model. Emotion Review 11(3):247–261. Alexithymia is the common downstream phenotype of multiple possible upstream impairments.

Six different upstream impairments — interoception, language, emotion concepts, predictive coding, attention/appraisal, symbolic elaboration — all converging on the same observable pattern. Different alexithymics can be on different routes. Some have multiple routes contributing simultaneously. The same TAS-20 score can mean six quite different things.

Route 1: Interoception

Bird/Murphy/Brewer lineage; Brewer 2016 / Murphy 2018 / Brewer 2021

Failure of detection, integration, or representation of bodily signals. Anterior insula, ACC, salience network.

Level
Afferent / network

Route 2: Language / labelling

Lieberman 2007; Hobson 2019

Failure in the verbal-symbolic system that maps physiological-conceptual states onto words. Left lateral PFC, affect-labelling pathway.

Level
Symbolic

Route 3: Emotion concepts

Barrett 2017 constructed-emotion frame

Impoverished conceptual system. Low emotional granularity. mPFC, default-mode network.

Level
Conceptual / categorisation

Route 4: Predictive processing

Quattrocki & Friston 2014; oxytocinergic tone

Atypical precision-weighting of interoceptive prediction errors. Salience network plus modulatory systems.

Level
Computational / active inference

Route 5: Attention / appraisal

Preece 2017 attention-appraisal model

Failure at attention or appraisal stage of emotion regulation. Executive control networks.

Level
Conscious-process / cognitive-behavioural

Route 6: Symbolic elaboration

Sifneos 1972; Taylor & Bagby 2024 defense

Failure of psychodynamic-style symbolic mentalisation of affect. The original psychosomatic frame — still actively defended.

Level
Developmental / functional
Therapeutic implication. Different alexithymics can sit at different routes. The same alexithymic can have impairments at multiple routes. The "right" intervention is route-specific:
  • Detection-deficit profile → interoceptive amplification (MABT-style scaffolded interoceptive work)
  • Concept/language deficit → granularity / affect-labelling work; Lieberman 2007 RVLPFC pathway
  • Attention deficit → attention-deployment training; CBT/DBT-style work
  • Precision-weighting deficit → oxytocin (preliminary) / scaffolded inference work
  • Symbolic-elaboration deficit → long-term psychodynamic / mentalisation-based therapy

No one-size-fits-all alexithymia therapy because no one-size-fits-all alexithymia mechanism. The user’s natural question — “where does my alexithymia live in the brain?” — has, on this account, no single answer. It depends on which upstream route is implicated.

Sources: Hobson H, Brewer R, Catmur C, Bird G. (2019) Emotion Review 11(3):247–261. SAGE DOI 10.1177/1754073919838528.
