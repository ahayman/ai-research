The Seven Heterogeneity Axes — Map of Subtype Distinctions

Alexithymia is not one thing. It fractures along seven partially independent axes. The same person can be at different positions on each. Treat this as a navigation toolkit, not a diagnostic algorithm.

Axis 1
TAS-20 Facets
DIF (identifying), DDF (describing), EOT (externally-oriented thinking). DIF & DDF correlate; EOT often dissociates and replicates poorly cross-culturally.
DIF/DDF strong EOT factor unstable
Axis 2
Bermond Type I / Type II
Cognitive-affective 2x2: Type I (low both, "affectless"); Type II (high affective + low cognitive, "feelings present but illegible"). Dimensions robust; categorical types weak.
Dimensions robust Types weak
Axis 3
Primary / Secondary / Organic
Etiological taxonomy. Primary = lifelong, dispositional. Secondary = trauma/stress/socialized. Organic = post-lesion, anterior insula. Useful clinical heuristic.
Heuristic, not validated
Axis 4
LPA / Cluster Subtypes
Originally proposed as four classes (Chen 2011/2017). Recent LPA/FMM work yields severity-distinguished 2-3 class solutions. Severity continuum better-fit than discrete classes.
Four-class scheme contested Severity continuum supported
Axis 5
State vs Trait
11-year stability is high (Tolmunen, N=3,083). But scores fluctuate with depression severity, acute distress. Trait core + state component. Therapy can produce measurable change.
Trait stability strong
Axis 6
Interoceptive Split
Detection-deficit, parsing-deficit, hypervigilance-with-mislabeling. Multiroute synthesis (Hobson 2019): interoception is one route, language is another. Each implies different therapy.
Multiroute synthesis solid Subtype-matched therapy untested
Axis 7
Cognitive vs Affective (BVAQ)
The most-replicated structural finding in alexithymia measurement. Cognitive (identifying/verbalizing/analyzing) and Affective (fantasizing/emotionalizing) dimensions can dissociate.
Most replicated structure

Sources: Bagby 1994, Schroeders 2022 (TAS-20); Vorst & Bermond 2001 (BVAQ); Messina 2014 (etiology); Schaeuffele 2021, Hogrefe FMM 2025 (LPA); Tolmunen 2011/2017 (stability); Brewer/Bird 2016, Hobson 2019, Aaron 2020 (interoception).
