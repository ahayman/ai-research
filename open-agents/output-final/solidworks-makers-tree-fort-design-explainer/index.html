SolidWorks Makers: Designing a Tree Fort from Phone Scan to Lumber List

Last updated: March 2, 2026 · 25 min read · Beginner level

Overview #

SolidWorks is professional 3D design software used by engineers to design everything from consumer electronics to aircraft parts. Through the "Makers" subscription, Dassault Systèmes (the company behind SolidWorks) offers this same software to hobbyists and personal users at a fraction of the professional cost — $48 per year or $15 per month.

The Makers subscription runs on a platform called 3DEXPERIENCE PlatformDassault Systèmes' cloud-based ecosystem that hosts SolidWorks and related apps, which is an enterprise cloud ecosystem designed for large companies with hundreds of employees in different roles. This is why the dashboard can feel overwhelming when you first log in — you're looking at an interface built for project managers, simulation engineers, and manufacturing teams. As a maker building a tree fort, you only need a small handful of those apps.

This guide is designed for Mac users using xDesign in the browser. The entire workflow — from importing your tree scan, to designing the structure with real lumber dimensions, to generating a bill of materials — happens in your web browser with no software installation required.

What is covered

  • Which subscription tier and apps to use
  • How to navigate the confusing 3DEXPERIENCE dashboard
  • How to 3D scan your tree and import the scan
  • How to design a structure using real lumber profiles
  • How to generate a cut list and bill of materials

Prerequisites #

Before beginning this project, you need:

  • A SolidWorks xDesign for Makers subscription ($9.99/month or included in the $48/year tier — see below)
  • A Mac (or any computer) with a modern web browser (Chrome, Safari, or Edge)
  • An internet connection (xDesign is cloud-based and requires connectivity)
  • A smartphone with a 3D scanning app installed
  • A tree you want to build around

No prior CAD experience is required. This guide assumes you are starting from scratch.

How to Scan the Tree #

  1. Choose the right conditions: Overcast days work well. Harsh shadows reduce scan quality.
  2. Focus on structure, not foliage: Scan the trunk and major branch crotches (where branches split from the trunk). Leaves and twigs are not useful data.
  3. Walk slowly in a circle around the trunk at the height where the fort platform will be built. Keep the camera pointed at the trunk.
  4. Scan from multiple heights if possible: ground level, intended platform level, and above.
  5. Export as OBJ format: OBJ handles large file sizes better than STL. File sizes typically range from 50 to 200 MB for a full tree trunk scan.

What the Scan Provides

The scan gives you the shape and dimensions of the tree trunk and main branches. Inside xDesign, this becomes a visual guide — a 3D ghost of the tree that you design your fort structure around. You can see exactly where the trunk is wide, where branches emerge, and where there's room for beams and platforms.

Cleaning Up the Scan Before Import #

Since xDesign does not include the desktop ScanTo3D Mesh Prep Wizard, you should clean up your scan before importing it. The goal is to remove unwanted data (ground, surrounding objects, leaves) and simplify the mesh so xDesign can handle it smoothly.

Option A: Clean up in the scanning app

Both Polycam and Scaniverse have built-in mesh editing tools. Before exporting:

  • Trim unwanted areas (ground, surrounding objects)
  • Delete stray geometry and noise
  • Crop the scan to focus on just the trunk and major branches

Option B: Use MeshLab (free, Mac-compatible)

MeshLab is a free, open-source mesh processing tool that runs on Mac. Use it to:

  • Remove noise: Filters → Cleaning and Repairing → Remove Isolated Pieces
  • Delete unwanted geometry: Select regions with the selection tool and delete them
  • Simplify the mesh: Filters → Remeshing → Simplification (reduce polygon count while preserving shape)
  • Export as clean OBJ: File → Export Mesh As → OBJ
Tip: The cleaner your mesh is before importing into xDesign, the better the performance. Aim to remove at least the ground plane and any objects that are not the tree trunk itself.

Part 4: Importing the Scan into xDesign #

With the scan cleaned up and exported as an OBJ or STL file on your computer, the next step is bringing it into xDesign.

Step 1: Open xDesign

  1. Log into the 3DEXPERIENCE platform in your browser
  2. Open xDesign from the Compass or dashboard
  3. Create a new design or open an existing one

Step 2: Import the Mesh File

  1. Go to File → Import
  2. Select "From local computer"
  3. Choose your cleaned OBJ or STL file from your Mac
  4. The mesh bodyA 3D surface made of triangles — the format that phone scans produce will appear in the design space
Note: Unlike the desktop SOLIDWORKS ScanTo3D add-in, xDesign does not have a Mesh Prep Wizard. This is why cleaning up the scan before import (see Part 3) is important. xDesign imports the mesh directly as-is.

Step 3: Use as a Reference Body

For this project, you do not need to convert the mesh into a solid body. Converting scans to solids is a complex reverse-engineering process used when you need to manufacture an exact replica of a scanned object. For a tree fort, the mesh serves purely as a visual reference bodyAn imported mesh used as a visual guide for designing around (not converted to a solid).

Keep the mesh in place and design your fort structure around it. You can:

  • Snap sketch points to mesh vertices (for precise alignment)
  • Create reference planes at specific heights to mark key levels (fort floor level, railing level, roof attachment)
  • Use the mesh as a visual guide to see where the trunk is wide, where branches emerge, and where there is room for beams and platforms

Part 5: Understanding Frame Design (3D Structure Creator) #

In xDesign, the 3D Structure Creator (also called Frame Design) is the tool for designing structures made from extruded profiles. It is the browser-based equivalent of the desktop SolidWorks "Weldments" toolset. The concept is the same: sketch paths, apply structural member profiles, and auto-generate a cut list. Despite the desktop feature's name implying metal welding, these tools work with any material that has a consistent cross-section — including wood lumber.

Figure 4: The frame design workflow: sketch centerlines (Step 1), apply lumber cross-section profiles (Step 2), and trim joints to get a finished structure with an auto-generated cut list (Step 3).

How Frame Design Works #

The Frame Design workflow has three stages:

  1. Sketch the framework: Create a 3D SketchA sketch mode where lines and geometry can exist in full 3D space (not limited to a flat plane) wireframe where each line represents the centerline of a piece of lumber. Vertical lines become posts, horizontal lines become beams and joists.
  2. Apply profiles: Use the Structural MemberA profile (cross-section) extruded along a sketch path — e.g., a 2x4 profile along a line becomes a 2x4 board tool in the 3D Structure Creator to apply a cross-section profile to each sketch line. The profile is extruded along the line's full length. A 2x4 profile applied to a 6-foot line produces a 6-foot-long 2x4.
  3. Trim joints: At intersections where two members meet, use the Trim/Extend tool to cut one member to fit against another — creating butt joints, miter cuts, or coped joints.

Lumber Profiles #

A "profile" in the 3D Structure Creator is a cross-section shape that gets extruded along your sketch lines. The tool comes with a built-in library of standard steel and aluminum profiles. For lumber, you need rectangular profiles matching actual lumber dimensions:

Nominal SizeActual DimensionsTypical Use

2x81.5" x 7.25"Floor joists, beams

Note that lumber nominal vs actual lumber sizeA "2x4" is nominally 2" x 4" but its actual milled dimensions are 1.5" x 3.5" (2x4) differ from actual dimensions (1.5" x 3.5"). Your profiles should use actual dimensions for accurate design.

Getting Lumber Profiles in xDesign

The 3D Structure Creator's built-in profile library includes standard rectangular tube and solid sections. For lumber, you have two approaches:

Option A: Use built-in rectangular profiles

The built-in profile library includes rectangular sections that can approximate lumber dimensions. Select the closest standard rectangular profile and adjust the dimensions to match actual lumber sizes (e.g., 1.5" x 3.5" for a 2x4).

Option B: Create a custom rectangular profile

  1. In xDesign's 3D Structure Creator, create a new sketch profile
  2. Draw a rectangle using actual lumber dimensions (e.g., 1.5" x 3.5" for a 2x4)
  3. Position the pierce pointThe point on a profile where it aligns to the sketch path (the origin of the profile sketch) at the center or corner of the rectangle, depending on how you want the lumber to align to your sketch paths
  4. Save the profile for reuse in your project
Note: The desktop version of SolidWorks uses custom Lib Feat Part (.sldlfp)The desktop-only SolidWorks file format for saved weldment profiles files for weldment profiles. This format is desktop-only. In xDesign, you work with the built-in profile library or create custom sketch profiles directly within the Structure Creator.

Phase 1: Support Posts

Support posts are the vertical members that hold up the platform.

  1. Create a new 3D Sketch in xDesign
  2. Draw vertical lines from ground level up to the intended platform height at each post location
  3. Position the posts around the tree, referring to the imported mesh to see where the trunk and branches are
  4. Close the sketch
  5. Open the 3D Structure Creator (Frame Design) tool
  6. Select a rectangular profile matching your lumber size — "4x4" (3.5" x 3.5") or "6x6" (5.5" x 5.5")
  7. Select the vertical sketch lines
  8. Click OK — the posts appear as 3D lumber pieces
Design consideration: Posts can be mounted directly in the ground (set in concrete footings), attached to the tree trunk with lag bolts, or standing on ground-level post anchors. Leave 2 to 4 inches of clearance between the tree trunk and any structural member — trees grow and sway in wind.

Phase 2: Platform Frame (Beams and Joists)

The platform frame consists of horizontal members that form the floor structure.

  1. Create a new 3D Sketch
  2. Draw horizontal lines connecting the tops of the posts to form the perimeter beams
  3. Add parallel lines between the perimeter beams for floor joists (typically spaced 16 inches on center)
  4. Leave an opening around the tree trunk — the platform frame should not contact the tree
  5. Apply 2x8 or 2x10 profiles to the perimeter beams (these carry the most load)
  6. Apply 2x6 or 2x8 profiles to the floor joists

Use the Trim/Extend tool at joints where joists meet beams. This creates realistic butt joints.

Phase 3: Decking (Floor Surface)

For the deck surface, you have two approaches:

Approach A (Detailed): Create closely spaced parallel sketch lines across the top of the joists and apply 5/4x6 or 2x6 deck board profiles. This represents each individual board and generates an accurate board count in the cut list.

Approach B (Simplified): Create a simple extruded solid representing the deck surface. This is faster to model but won't generate individual board counts in the BOM.

Phase 4: Railings and Walls

Safety railings are critical for a tree fort.

  1. Create vertical sketch lines from the platform edge upward to railing height (36 to 42 inches is standard for deck railings)
  2. Add horizontal lines for top and mid rails
  3. Add vertical lines between rails for balusters (spaced no more than 4 inches apart per standard building codes)
  4. Apply 4x4 profiles to corner posts, 2x4 profiles to rails, and 2x2 profiles to balusters

Phase 5: Roof Structure (Optional)

If your fort includes a roof:

  1. Create angled lines from the tops of the wall posts upward to a ridge line
  2. Add the ridge beam as a horizontal line at the peak
  3. Add collar ties (horizontal cross-members) for structural rigidity
  4. Apply 2x4 or 2x6 profiles to rafters and 2x6 to the ridge beam

Phase 6: Access (Ladder or Stairs)

  1. Create two angled parallel lines from ground level up to the platform edge — these are the stringers
  2. Add horizontal lines between the stringers for rungs or treads
  3. Apply 2x4 profiles to stringers and 2x4 or 1x4 profiles to rungs

Working Around the Tree

Throughout the design process, the imported tree mesh serves as your guide:

  • Position structural members to avoid the trunk and major branches
  • Use reference planes at key heights to keep everything aligned
  • Leave growth gaps — the opening around the trunk should be larger than the current trunk diameter to account for tree growth
  • Consider a floating platform design: instead of bolting directly to the tree, the platform rests on posts and uses sliding brackets where it meets the tree. This allows the tree to move independently of the structure.

The Cut List (Automatic)

When you create structural members using the 3D Structure Creator, xDesign automatically generates a Cut List from your structural members.

The Cut List organizes every structural member body into groups. Members with identical profiles and lengths are grouped together and counted. Each Cut List Item includes:

  • Item number — auto-assigned identifier
  • Description — the profile name (e.g., "2x4 Lumber" or "4x4 Post")
  • Quantity — how many identical pieces exist
  • Length — the measured length of the member

Creating a Drawing with BOM Table

To produce a printable shopping list, use xDrawing (included in xDesign as of 2025):

  1. Open xDrawing from within xDesign or from the dashboard
  2. Create a new drawing and insert a view of your design
  3. Insert a BOM table into the drawing
  4. The table automatically fills with all cut list data from your structural members

You can customize the table by:

  • Adding columns (wood species, treatment type, estimated cost)
  • Sorting by lumber size or by structural group
  • Adding total quantity summaries

Exporting to CSV

To create a portable shopping list:

  1. From your BOM table in xDrawing, export the data
  2. Choose CSV format
  3. Save the file to your computer

This spreadsheet becomes your lumber yard shopping list, with columns for size, length, quantity, and any custom properties you've added. Open the CSV in any spreadsheet app (Numbers, Google Sheets, Excel) to organize and print.

Common Misconceptions #

"Frame Design / Structure Creator is only for metal"

Despite the historical name "Weldments" in the desktop version, the 3D Structure Creator in xDesign works with any material that has a consistent cross-section profile. The same tools that design steel tube frames also design lumber structures, aluminum framing, or PVC pipe assemblies. The tool was originally developed for welded metal structures but has since become a general-purpose framing tool.

"I need to convert my scan to a solid body"

For this project, converting the tree scan to a solid is unnecessary and adds significant complexity. The mesh serves as a visual reference — a 3D guideline showing where the tree is so you can design around it. Solid body conversion (reverse engineering) is used when you need to manufacture an exact replica of a scanned object.

"xDesign can't do real structural design"

xDesign includes the 3D Structure Creator with full frame design capabilities: structural member profiles, automatic trimming at joints, and automatic cut list generation. For a project like a tree fort, xDesign provides all the structural design tools you need, entirely in the browser. The desktop SOLIDWORKS Connected application offers additional features (like custom .sldlfp profile files and the ScanTo3D Mesh Prep Wizard), but they are not required for this workflow.

"I need to understand all the apps on the dashboard"

The 3DEXPERIENCE dashboard shows many apps because the platform serves diverse professional roles. As a Maker building a tree fort on a Mac, you need three apps: xDesign (main design tool with 3D Structure Creator), xDrawing (for BOM tables), and 3DDrive for file storage. Everything else can be ignored.

Tips and Best Practices #

Scanning Tips

  • Scan on overcast days for best results — direct sunlight creates harsh shadows that confuse photogrammetry
  • Walk slowly and overlap your path — the app needs to see each surface from multiple angles
  • Focus on the trunk and major branches only — leaves and small twigs add noise without useful structural information

Design Tips

  • Start simple: get the posts and platform frame right before adding railings and roof
  • Use 3D sketch relations and constraints to keep members aligned to reference planes
  • xDesign auto-saves to the cloud, so you don't need to worry about manual saves — but be aware that your work is always syncing
  • Since xDesign requires internet, make sure you have a reliable connection before long design sessions
  • Use Section Views to see inside the structure and check clearances around the tree

BOM Tips

  • Add custom properties to your structural member profiles (wood species, treatment type) so they appear in the BOM automatically
  • Round cut lengths up to the nearest standard lumber length (6', 8', 10', 12') for purchasing — you'll cut pieces to final length on site
  • Export your BOM to CSV and open in a spreadsheet app (Numbers, Google Sheets) to group by lumber size for easier ordering at the lumber yard

Summary #

Key Points

  1. The SOLIDWORKS xDesign for Makers subscription provides all the tools Mac users need for this project: direct mesh import, Frame Design (3D Structure Creator) for lumber design, and automatic Cut List for BOM generation — all in the browser.
  2. The 3DEXPERIENCE dashboard contains many apps, but you only need three: xDesign with 3D Structure Creator (main design), xDrawing (for BOM tables), and 3DDrive (cloud storage). Navigate using the Compass (top-left) or Search bar.
  3. Scan your tree with a phone app like Polycam or Scaniverse, clean up the mesh in the scanning app or MeshLab, export as OBJ, and import into xDesign as a reference body — you design around it, not from it.
  4. The 3D Structure Creator (Frame Design) turns 3D sketch lines into structural lumber by extruding cross-section profiles (2x4, 4x4, etc.) along paths. Despite the desktop name "Weldments," these tools work for wood.
  5. The Cut List auto-generates from your frame design. Export it to CSV via xDrawing for a ready-made shopping list.

Structural MemberA profile (cross-section) extruded along a sketch path — e.g., a 2x4 profile along a line becomes a 2x4 board
Pierce PointThe point on a profile where it aligns to the sketch path (the origin of the profile sketch)

Mesh BodyA 3D surface made of triangles — the format that phone scans produce

3D SketchA sketch mode where lines and geometry can exist in full 3D space (not limited to a flat plane)

3DDriveCloud storage included with the Makers subscription (25 GB)
PhotogrammetryThe technique of creating 3D models from multiple overlapping photographs
LiDARLight Detection and Ranging — a laser-based sensor in iPhone Pro models that measures distances for more accurate 3D scans
